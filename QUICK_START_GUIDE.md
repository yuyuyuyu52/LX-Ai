## 🛠️ 开发者指南

### CSS 样式系统
主要样式文件: `static/css/zen-style.css`

**核心组件类**:
```css
.zen-btn          /* 按钮系统 */
.zen-card         /* 卡片组件 */
.zen-form-control /* 表单控件 */
.zen-badge        /* 徽章标签 */
.zen-notification /* 通知系统 */
```

**动画效果类**:
```css
.gentle-float     /* 轻柔浮动 */
.fade-in          /* 淡入效果 */
.slide-up         /* 上滑效果 */
```

### 颜色变量
```css
:root {
  --zen-bg: #FDF9F3;
  --zen-primary: #3E2C1D;
  --zen-secondary: #A89E92;
  --zen-accent: #C8B89E;
  --zen-decoration: #E7DAC6;
}
```

### 添加新页面
1. 继承 `base.html` 模板
2. 使用 `zen-` 前缀的CSS类
3. 遵循东方美学设计原则
4. 添加适当的动画效果
